# Task 6 - Password Strength Evaluation

## Objective
The objective of this task is to understand what makes a password strong and evaluate different passwords using an online password strength checker.

## Tool Used
- PasswordMeter (https://passwordmeter.com)

## Steps Performed
1. Created five passwords with different levels of complexity.
2. Tested each password using PasswordMeter.
3. Recorded the password strength and observations.
4. Compared the results.
5. Documented best practices for creating secure passwords.

## Passwords Tested

| Password | Strength |
|----------|----------|
| task567 | Very Weak |
| pass@655 | Weak |
| pass@gjslrhui | Fair |
| pass@gjslrhuiuhe | Strong |
| pass@gjslr126627 | Very Strong |

## Key Learnings
- Longer passwords are more secure.
- Using uppercase, lowercase, numbers, and special characters improves password strength.
- Avoid common words and predictable patterns.
- Strong passwords help protect against brute-force and dictionary attacks.
- Multi-Factor Authentication (MFA) adds an extra layer of security.
